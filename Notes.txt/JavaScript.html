What is JavaScript?

  • JavaScript is a programming language used to make websites interactive.
  • It allows you to create dynamic content like buttons, animations, and forms.
  • It's widely used in web development and is easy to learn.

What is the History of JavaScript?

  • JavaScript was created by Brendan Eich in 1995 while he was working at Netscape.
  • It was initially called LiveScript but later renamed JavaScript.
  • It is mainly used as a scripting language in web browsers, where it interacts with web pages, though it doesn't handle input
    or output directly.
  • The browser provides the tools for interacting with the outside world.

JavaScript Versions:

  • ES1 (1997): First official version of JavaScript, standardized as ECMAScript.
  • ES3 (1999): Added regular expressions, try/catch for error handling, and better string manipulation.
  • ES5 (2009): Introduced "strict mode," JSON support, Array methods (forEach, map, filter), and more.
  • ES6 (2015): A significant update, introducing let/const, arrow functions, classes, promises, template literals, and modules.
  • ES7 (2016): Added Array.prototype.includes and the exponentiation operator (**).
  • ES8 (2017): Introduced async/await, Object.entries, and Object.values.
  • ES9 (2018): Added features like rest/spread properties for objects and asynchronous iteration.
  • ES10 (2019): Added flat(), flatMap(), optional catch binding, and more.
  • ES11 (2020): Introduced dynamic import, nullish coalescing (??), optional chaining (?.), and BigInt.
  • ES12 (2021): Brought new features like replaceAll() for strings, and numeric separators.

Features of JavaScript:

  • Supported by all web browsers: Every major web browser (like Chrome, Firefox, Safari) can run JavaScript because it has built-in tools for it.
  • Similar to C language: JavaScript's structure and rules are similar to the C programming language, making it an organized language.
  • Flexible with data types: JavaScript is a "weakly typed" language, which means it can automatically change the type of data based on what you’re doing.
  • Object-oriented but different: JavaScript is object-oriented, but it uses prototypes instead of traditional classes for inheritance.
  • Light and fast: It's a lightweight language and runs immediately without needing to be compiled.
  • Case-sensitive: Uppercase and lowercase letters matter in JavaScript, so name and Name would be treated as different things.
  • Works on many operating systems: JavaScript runs smoothly on various systems like Windows, macOS, and others.
  • Gives control over the browser: It allows users to interact with web browsers easily (like opening pop-ups or changing what’s displayed on the screen).

Applications of JavaScript

JavaScript is used to make websites interactive. It can:

  • Check user input: Like making sure a form is filled correctly.
  • Create dynamic menus: Menus that change on the fly.
  • Show the current date and time: Displaying real-time information.
  • Display pop-ups and dialog boxes: Such as alerts, confirmations, or prompts.
  • Create clocks and other real-time displays: For tracking time and updating content in real time.

How to Add JavaScript in HTML Document?

To add JavaScript to an HTML document, you can use a few methods. You can either write JavaScript directly in the HTML
file or link to an external JavaScript file.

Inline JavaScript

We can write JavaScript code directly inside the HTML element using the onclick, onmouseover, or other event handler attributes.

Internal JavaScript (Within script Tag)

We can write JavaScript code inside the script tag within the HTML file.
This is known as internal JavaScript and is commonly placed inside the head or body section of the HTML document.

1. JavaScript Code Inside head Tag

Placing JavaScript within the head section of an HTML document ensures that the script is
loaded and executed as the page loads. This is useful for scripts that need to be initialized before the page content is rendered.

2. JavaScript Code Inside body Tag

JavaScript can be added inside the body section of an HTML page. It's common to place it at
the end of the body so that the content of the page loads first. This is helpful if the script
needs the whole page to be loaded before it runs.

External JavaScript (Using External File)

For bigger projects or when you want to use the same script in multiple HTML files, you can put your
JavaScript code in a separate .js file. You then connect this file to your HTML document by using the src attribute inside a scripts tag.

JavaScript Syntax

JavaScript syntax is the set of rules that defines how to write and structure JavaScript code. It includes things like statements,
variables, functions, operators, and how to control the flow of a program.
